# Uber Eats Clone

### Technologies Used
* React Native
* Expo
* React Navigation 
* Redux
* Place Autocomplete Places API
* Lottie
* Yelp API - **[https://saurav.tech/NewsAPI/](https://www.yelp.com/developers)**

Note: no design library is used. The entire UI of this application is built from the ground up.
